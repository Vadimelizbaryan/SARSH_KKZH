# MAINFORM Android

Android application shell for the SARSH department workflow.

Current release APK:

- `android/releases/MAINFORM.apk`

Project source:

- `android/MAINFORM/`

Build notes:

- .NET for Android (`net8.0-android`)
- Android SDK platform 34
- Java 17

The app opens the existing deployed department pages in a native Android WebView,
lets the user choose a department, open the main page, refresh, upload/take a
photo, and work directly against the current server endpoints.
