# Распознавание Фото Бланков

В проект уже добавлена функция загрузки фото на страницах отделений.

Как это работает:

1. Откройте страницу нужного отделения.
2. В блоке `Загрузка фото бланка` выберите фотографию бланка.
3. Нажмите `Распознать`.
4. Программа подставит распознанные цифры в ячейки только локально.
5. Проверьте значения, при необходимости исправьте вручную.
6. Нажмите обычную кнопку `Сохранить`, чтобы отправить строку в главный файл.

Что распознаётся в первой версии:

- дата бланка;
- только верхняя числовая таблица;
- рукописный текст в нижней части бланка сейчас игнорируется.

Что уже защищено:

- после распознавания автоотправка временно ставится на паузу;
- до сохранения пользователь может спокойно поправить ячейки вручную;
- при попытке обновить страницу с несохранённым распознанным черновиком система предупреждает.

Что нужно для онлайн-работы:

1. В секрете функции `sharsh-sync` в Supabase добавить `OPENAI_API_KEY`.
2. При желании добавить `OPENAI_PHOTO_MODEL`.
   По умолчанию используется `gpt-5.4-mini`.
3. Заново задеплоить функцию `supabase/functions/sharsh-sync/index.ts`.

Если серверная часть ещё не задеплоена, кнопка распознавания в интерфейсе появится, но само распознавание работать не будет.
