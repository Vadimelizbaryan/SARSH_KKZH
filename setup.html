Подключение к серверу

Сюда вставляется только публичный ключ клиента Supabase. Секретный service_role здесь не нужен.

Для этого проекта используйте именно Legacy API Keys → anon. Не вставляйте service_role.

Готовые ссылки

После проверки подключения используйте эти ссылки для главной страницы и отдельных отделений.

Как настроить

  1. В Supabase откройте настройки проекта и скопируйте Project URL и Legacy anon key.
  2. Вставьте их в поля слева и нажмите Проверить подключение.
  3. Если подключение работает, нажмите Сохранить в браузере.
  4. Откройте главный файл или скопируйте ссылки отделений.
  5. Если что-то пошло не так, нажмите Вернуть локальный режим.

Настройки сохраняются только в этом браузере. Файлы проекта и данные отделений не меняются до сохранения на сервере.
